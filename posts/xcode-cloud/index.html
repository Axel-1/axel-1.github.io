Apple a annoncé un nouvel outil pour les développeurs : Xcode Cloud

2021-06-07
#iOS  #Veille 

Le nouvel outil Xcode Cloud permet au développeur de faciliter le CI/CD sur les projets iOS. En effet, Xcode Cloud offre la possibilité de réaliser des tests sur tous les types d’appareils compatibles avec le projet, directement dans le cloud.
