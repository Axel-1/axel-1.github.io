Annonce de Flutter 2

2021-03-03
#Android  #Veille 

Flutter 2 est disponible avec l’ajout du support pour le Web permettant de réaliser une application mobile et web en utilisant le même code source. Cela tout en conservant une interface différente et adapté pour chaque plateforme. Il permet également de respecter les règles de design de chaque appareil avec toujours le même code source.
