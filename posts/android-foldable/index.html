Google annonce les nouveautés liées aux appareils avec écran pliant

2021-05-18
#Android  #Veille 

Avec l’arrivée des smartphones avec écran pliant, Google annonce différentes fonctionnalités d’Android pour les développeurs afin de mieux gérer ce nouveau type d’appareil dans les applications. Également, la firme de Mountain View donne différentes directives sur comment enrichir son application pour ce type d’écran.
